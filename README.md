# Abacus Trainer Website

A modern, responsive website for the Abacus Trainer app - the most advanced abacus practice app powered by AI and gamification.

## Features

- **Modern Design**: Clean, professional layout with gradient backgrounds and smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: Smooth scrolling, hover effects, and interactive elements
- **Performance Optimized**: Debounced scroll events and optimized animations
- **Accessible**: Proper semantic HTML and keyboard navigation support

## File Structure

```
├── index.html          # Main HTML file with all sections
├── style.css           # Complete CSS with responsive design
├── script.js           # Interactive JavaScript features
├── images/             # Directory for images and screenshots
│   └── phone-mockup.png # Placeholder for app screenshots
└── README.md           # This file
```

## Sections Included

1. **Hero Section** - Main headline with CTA buttons and phone mockup
2. **Key Benefits** - Three main value propositions
3. **How It Works** - Step-by-step process
4. **Features Overview** - Detailed feature list
5. **Who It's For** - Target audience breakdown
6. **Plans & Pricing** - Pricing tiers and options
7. **Testimonials** - Social proof and reviews
8. **Final CTA** - Strong closing call-to-action
9. **Footer** - Contact info and links

## Customization

### Images
- Replace `images/phone-mockup.png` with actual app screenshots
- Add any additional images to the `images/` directory
- Update image paths in HTML if needed

### Colors
The website uses a modern color scheme:
- Primary: #6366f1 (Indigo)
- Secondary: #10b981 (Emerald)
- Background: #f8fafc (Light gray)
- Text: #1f2937 (Dark gray)

### Content
- Update contact email in footer
- Modify pricing information
- Add real testimonials
- Update feature descriptions
